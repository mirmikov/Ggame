# Легальные ассеты

Текущий MVP не требует внешних изображений: арена, башни, юниты и эффекты сделаны на CSS.

## Используемые ассеты

Юниты, башни, ракеты и взрывы используют SVG из [OpenMoji](https://openmoji.org/):

- источник: [официальный репозиторий OpenMoji](https://github.com/hfg-gmuend/openmoji);
- файлы: `frontend/public/assets/openmoji`;
- лицензия: [CC BY-SA 4.0](https://creativecommons.org/licenses/by-sa/4.0/);
- авторство: OpenMoji, HfG Schwäbisch Gmünd.

Цветовые варианты команд создаются CSS-фильтрами; оригинальные SVG сохранены локально без зависимости от CDN.

Рекомендуемые источники для дальнейшего развития:

- [Kenney](https://kenney.nl/assets) — игровые наборы преимущественно CC0.
- [OpenGameArt](https://opengameart.org/) — проверяйте лицензию каждого набора.
- [itch.io free game assets](https://itch.io/game-assets/free) — используйте фильтр лицензий и сохраняйте сведения об авторстве.
- [Google Fonts](https://fonts.google.com/) — шрифты с открытыми лицензиями.
- [Lucide](https://lucide.dev/) и [Heroicons](https://heroicons.com/) — открытые SVG-иконки.

Перед добавлением ассета сохраняйте ссылку, автора, версию лицензии и необходимые строки атрибуции.
