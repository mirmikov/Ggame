# План развития

## Следующий релиз

- Расширенный Terminal с редактором и несколькими тест-кейсами.
- Таймер каждого вопроса и автоматический неверный ответ по истечении времени.
- Переподключение игрока по токену и восстановление сессии.
- Настраиваемый баланс авто-боя и отдельный пакет `internal/game`.
- Панель ведущего с выбором набора вопросов.

## Production

- PostgreSQL, аккаунты, история матчей и рейтинг.
- Авторизация, rate limiting, серверная валидация всех действий.
- Масштабирование WebSocket через Redis/NATS.
- Изолированный sandbox для запуска кода.
- Тесты доступности, локализация, звуки и лицензированные ассеты.
- Unit-, integration- и browser-тесты для критических сценариев.
